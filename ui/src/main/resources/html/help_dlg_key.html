Each round of a Substitution-Permutation Network (SPN) includes a subkey. A subkey is a string of bits equal in length to the block size of the round. Each bit in the subkey is combined with the input bits to the round using some logical operator. In this application, the subkeys are processed using exclusive-OR (XOR).
