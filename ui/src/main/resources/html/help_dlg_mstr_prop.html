Help With Master Properties

The Master Properties Dialog is used to set global variables for use in the user-defined Substitution-Permutation Network (SPN). They are as follows:

Field Description
Block size The number of bits going into each round of the SPN cipher.
Num. rounds The number of rounds through which the data will iterate during the encryption process. The final round consists only of a Key, and this tool will not allow other components.
S-box size The number of bits in each bijective S-box for the cipher. All S-boxes in the SPN will be of equal size, and each will have input size equal to their output size. This value must evenly divide the block size.
